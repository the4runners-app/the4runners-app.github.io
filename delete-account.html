Suppression de compte Audio2Text

Si vous souhaitez supprimer votre compte Audio2Text et les données associées, vous pouvez effectuer la demande directement depuis l'application, dans l'onglet Compte, ou en nous contactant à l'adresse suivante :

Email de contact :
the4runners.app@gmail.com

Données supprimées

La suppression du compte entraîne la suppression des données associées au compte suivantes :

  • les informations du profil utilisateur ;
  • le portefeuille de crédits associé au compte ;
  • les transcriptions sauvegardées dans le cloud ;
  • la photo de profil éventuelle.

Données pouvant être conservées temporairement

Certaines données techniques ou journaux minimaux peuvent être conservés temporairement pour des raisons légales, comptables, de sécurité ou de prévention des abus.

Informations à fournir

Pour traiter votre demande par email, merci d'indiquer l'adresse email utilisée pour votre compte Audio2Text.
